# Twitter PinBased Oauth


## Licence 
MIT Licence

## Install

```shell
npm install -g twitter-pinbased-oauth
```
or
```shell
yarn global add twitter-pinbased-oauth
```

## Use

```
twitter-pinbased-oauth {consumerKey} {consumerSecretKey}

Please type PIN
```
